# Rock, Paper, Scissors (Vue 3 + Vite)

The Rock, Paper, Scissors game uses Vue 3 `<script setup>` SFCs, check out the [script setup docs](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup) to learn more.
